# 画面遷移図

| 出発画面 | アクション | 遷移先URL | 遷移先画面 |
|----------|------------|-----------|------------|
| (初期) | アクセス | `/` | S01: ISBN入力画面 |
| S01 | 確認ボタン押下 | `/books/confirm?isbn=...` | S02: 本登録確認画面 |
| S02 | 登録ボタン押下 | `/books/register` (POST) | S03: 本棚一覧画面 (リダイレクト) |
| S03 | 新規登録ボタン押下 | `/` | S01: ISBN入力画面 |
| S03 | 編集ボタン押下 | `/books/edit/{id}` | S03上のモーダル or 編集用別URL |
| S03 | 更新ボタン押下 | `/books/update` (POST) | S03: 本棚一覧画面 |
